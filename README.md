# python-data-automation-demo
# Python Data Automation Demo
Small, practical examples of reading, transforming, and exporting data with Python.

## Contents
- `01_read_csv.py` – Reads a CSV, cleans columns, writes JSON.
- `02_transform.py` – Basic aggregation and formatting.

## Why this repo?
Demonstrates Python fundamentals used in entry-level SWE roles:
- File I/O, data parsing, and simple transformations
- Clean, readable code with docstrings
- Reusable functions and clear structure
